Sales Dashboard – 📊
Project Overview 

An interactive business intelligence dashboard designed to track sales performance, customer behavior, profitability trends, and category-level insights for an e-commerce business.
The objective of this project was to transform raw transactional data into actionable insights that support data-driven decision-making across sales, marketing, and inventory operations.

Tools & Technologies Used 🛠️ 

• Power BI Desktop: Dashboard development, data modeling, and visualization

• Excel: Data source preparation and cleaning

• DAX (Data Analysis Expressions): Calculated metrics like profit margin, AOV, and repeat customer rate

• Power BI Features Used ⚡

• Cards & KPIs for key metrics

• Line, bar, and column charts for trend analysis

• Slicers for dynamic filtering by year, category, region, and segment

• Interactive dashboard elements (tooltips, drill-throughs)

Files Included 📁

📄 Sales_Data.xlsx – Raw dataset 

📊 Sales_Dashboard.pbix – Power BI dashboard file

📘 README.md – Project documentation with insights and instructions

Features of the Dashboard 🖥️

• Total Sales & Profit KPIs: Tracks year-wise performance and profitability

• Average Order Value & Profit Margin: Monitors transaction value trends and efficiency

• Repeat Customer Rate: Shows customer loyalty trends over time

• Category, Region, and Segment Analysis: Provides insights on performance across different dimensions

• Product Analysis: Highlights top-performing products to focus inventory and marketing strategies

• Monthly Trend Line Chart: Captures seasonal peaks, volatility, and growth patterns

• State-Level Analysis: Identifies high-performing regions and opportunities for expansion

• Interactive Filtering: Allows users to dynamically explore data by year, region, category, and segment

Key Insights (Summary) 

• Sales, profit, and AOV show a steady increase from 2012 to 2014

• Repeat customer rate fluctuates, dipping in 2013 but recovering in 2014

• Profit margin jumps in 2014, indicating improved efficiency or higher-margin sales

• Technology category is declining, Office Supplies are growing, and Furniture is volatile

• California and New York dominate sales, highlighting regional concentration

• Top 5 products consistently contribute the most revenue

• Both B2B (Corporate, Home Office) and Consumer segments show strong repeat behavior
